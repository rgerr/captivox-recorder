# Edition and Licensing Note

## Donationware edition

Captivox Recorder v0.9.0-beta1 is distributed as donationware. All of the functionality described in the current beta — including recording, transcription, structured insights, Microsoft 365 integration, local model selection and diagnostics — is available without payment. There is no time limit or feature gating in this edition.

If you find Captivox useful and want to support ongoing development, please consider making a donation via the link in the README. Donations help fund continued testing, feature development and maintenance.

## Paid features in the future

We anticipate that future releases may include premium features layered on top of the free core. Examples could include:

- cross-session search and advanced recall
- automation workflows (e.g. creating tasks, reminders or calendar events from transcripts)
- integration with additional AI models and enrichment services

These paid features would be clearly separated from the free core and would never compromise the local-first design or restrict access to your data.

## Source code

The Captivox Recorder source code is not published at this time. The public repository contains the installer, documentation and issue tracking. Developer discussions and implementation details are managed privately. This policy may change in the future.
